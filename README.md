# EMS-Employee-management-system-
🗂️ EMS Portal — Employee Management System

A role-based task management web app where Admin can create & assign tasks to employees, and employees can accept & track their work in real time.

🔐 Two Roles:
- Admin — Assign tasks with title, description, due date & category. Monitor all employees' task progress.
- Employee — View assigned tasks, accept them, and mark as completed.

⚙️ Tech Stack:
- React.js
- Tailwind CSS
- JavaScript
- LocalStorage (as persistent storage)

✨ Features:
- Role-based login system
- Real-time task status tracking (New → Active → Completed → Failed)
- Clean dark UI
- No backend required — fully runs in the browser
